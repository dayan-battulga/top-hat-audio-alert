If you want a tighter manifest or store summary later, use:

## Recommended detailed description

Top Hat Audio Alert is an unofficial Chrome extension that helps you notice new Top Hat classroom questions and participation prompts by playing a local sound on supported Top Hat classroom pages.

Choose one of the built-in sounds or upload your own MP3, set the volume, preview sounds, and pause alerts anytime from the popup. Your settings and optional custom audio stay in your browser's local extension storage.

The extension works on `https://app.tophat.com/e/*` while you are signed in to Top Hat. It reads the current classroom page only to detect when a new question or participation prompt becomes visible. It does not send Top Hat page data, settings, or uploaded MP3s to the developer or third parties.

Features:

- Detects new Top Hat questions and participation prompts on supported classroom pages
- Avoids replaying the same alert on harmless rerenders of the same item
- Includes 3 built-in sounds: Chime, Bell, and Pulse
- Supports optional custom MP3 uploads
- Lets you set volume, preview sounds, and pause alerts from the popup
- Stores settings locally in your browser

This project is not affiliated with or endorsed by Top Hat.
